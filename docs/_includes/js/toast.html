用法

调用$.toast()可触发提示框事件，可传入对象参数；亦可省略参数名，默认第一个是text,第二个type,第三个position,如： $.toast('button bottomright warning','warning','bottom right');

{% highlight js %} $('#toast-topleft').click(function(event){ event.preventDefault(); $.toast({text:'button topleft danger',type:'danger',position:'top left'}); }); $('#toast-top').click(function(event){ event.preventDefault(); $.toast({text:'button top success',posititon:'top',type:'success'}); }); $('#toast-topright').click(function(event){ event.preventDefault(); $.toast({text:'button topright warning',type:'warning',position:'top right'}); }); $('#toast-center').click(function(event){ event.preventDefault(); $.toast({text:'button center danger',type:'danger',position:'center'}); }); $('#toast-bottomleft').click(function(event){ event.preventDefault(); $.toast({text:'button bottomleft danger',type:'danger',position:'bottom left'}); }); $('#toast-bottom').click(function(event){ event.preventDefault(); $.toast({text:'button bottom success',type:'success',position:'bottom'}); }); $('#toast-bottomright').click(function(event){ event.preventDefault(); $.toast('button bottomright warning','warning','bottom right'); }); {% endhighlight %}

Options

名称 默认值 说明
position 'top' 显示位置，可选top,left,right,center,bottom,bottom,middle,默认为center middle,可省略写
type 'danger' 提示类型，可选danger,default,success,primary,warning,info
timeout 5000 自动隐藏时间
closeOnSelfClick true 是否允许点击自己控制弹出框关闭
texts '' 弹出框显示文字

事件

1.可直接调用$.toast('字符串');改变提示文字

2.可调用$.toast().show();$.toast().hide();手动控制提示框显示和消失

{% highlight js %} $.toast().show(); $.toast().hide(); {% endhighlight %}
