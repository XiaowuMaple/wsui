2.容易使用

包括js和css文件， 调价"data-step" 和 "data-intro" 到你的代码。

3.键盘 + 鼠标 控制

用鼠标, 或键盘 "←"控制后退, "→"控制前进, "ENTER" 和 "ESC" 控制退出.

5.快 & 小

仅需4 KB JavaScript 和 2.5 KB CSS (gzip).

6.跨浏览器兼容性

它兼容最近版本的火狐,Chrome和IE8，更好的兼容性需要进一步开发。
