禁用和激活状态

所有的链接都可以自己定义状态。加上 .disabled 会变成禁用样式，加上 .active 变成激活状态.

  • 上一页
  • 1 (current)
  • 2
  • 3
  • 4
  • 5
  • 下一页
{% highlight html %}
  • 上一页
  • 1 (current)
    • ...
{% endhighlight %}

你可以把激活/禁用状态下的链接换成 <span>，或者直接省略链接，这样可以保证用户无法点击。

{% highlight html %}
  • 上一页
  • 1 (current)
    • ...
{% endhighlight %}
