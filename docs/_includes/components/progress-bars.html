进度条 设计稿

用来显示进度，比如上传文件等.

不同浏览器的兼容性问题

进度条使用了CSS动画，所以在IE9以及低版本的火狐上无法看到动画效果。

基本用法

默认的样式.

60% Complete

60% Complete
{% endhighlight %}

不同大小

加上 .progress-lg 或者 .progress-sm 可以改变进度条的高度。

60% Complete

{% endhighlight %}

带标签

把 <span> 上的 .sr-only 类去掉，就可以显示出文案来。

{% endhighlight %}

在 .progress-bar-striped 上加上一个 .active 类就可以显示出滚动动画。IE9及以下的浏览器不支持。
