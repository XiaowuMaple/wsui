SUI

SUI 是一套基于 Bootstrap 的全栈开发工具包，包含丰富的组件库，同时提供完整的前端开发指导。

内置模板

SUI 针对后台系统中常见的页面提供了示例模板，下载后即可以直接套用在项目中，无需关注环境配置。

系统首页

由导航模块、轮播组件、活动 Banner 区块等展示性组件组成的常见门户首页。

表单页面

表单模板包含大量常见的表单域，适用于快速搭建后台系统中大量的表单页面。

列表页面

列表页面模板适用于快速构建可容纳大量数据展现的列表页面，内置了表格排序、分页等组件。

页面快速构建工具箱

SUI Toolkit 是一个专门为SUI提供的可视化编辑工具，即使没有任何前端知识，你也可以通过鼠标拖拽和点击轻松搭建一个漂亮的页面。对于产品经理和设计师，它是一个原型设计工具，可以快速搭建产品原型。对于程序员，它可以迅速搭建网页，生成高质量的HTML代码，并可以一键下载。赶紧来体验吧。
